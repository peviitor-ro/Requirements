SCRAPERE & AUTOMATIZARE
peviitor.ro este un proiect open source. Oricine poate contribui, oricine poate scrie cod. Cei care scriu un scraper alimentează baza de date de care se bucură mii de utilizatori aflați în căutarea unui loc de muncă.

Rolul scraperului

Un scraper este puntea dintre o sursă de locuri de muncă și motorul de căutare peviitor.ro. Rolul său este să extragă, transforme și trimită datele în baza de date centrală, unde devin disponibile pentru căutare.

Fără scrapere, nu există date. Fiecare scraper adaugă o nouă sursă de locuri de muncă, lărgind astfel acoperirea platformei și oferind mai multe opțiuni utilizatorilor.

Tehnologii

Nu există o singură tehnologie impusă. Un scraper poate fi scris în orice limbaj și cu orice unealtă, atâta timp cât respectă formatul de ieșire și protocolul de comunicare cu API-ul. Exemple de tehnologii folosite:

  • Python — cel mai comun, cu biblioteci precum Requests, BeautifulSoup, Scrapy, Selenium
  • PHP — folosit în ecosistemul existent al validatorului
  • Node.js / TypeScript — cu Puppeteer, Playwright, Cheerio
  • Shell scripting — pentru surse simple cu API public
  • AI / LLM — pentru extragere inteligentă din pagini nestructurate
  • Orice alt limbaj — atâta timp cât produce date în formatul corect

Flexibilitatea tehnologică încurajează contribuții din partea cât mai multor voluntari, indiferent de stack-ul cu care sunt familiarizați.

Cine poate scrie un scraper

Oricine. Nu trebuie să fii membru al asociației, nu trebuie să ceri permisiune. Tot ce ai nevoie este:

  • Să cunoști un limbaj de programare
  • Să ai un cont pe GitHub
  • Să urmezi formatul documentat al scraperului
  • Să trimiți un pull request

După review și testare, scraperul tău devine parte din platformă. Numele tău va apărea în lista de contribuitori.

De ce să scrii un scraper

Fiecare scraper adăugat înseamnă mai multe locuri de muncă disponibile pentru cei care caută. Contribuția ta are un impact direct și imediat:

  • Ajuți oameni — sute de persoane își găsesc locul de muncă prin peviitor.ro
  • Înveți — scraping-ul te învață HTML, rețele, parsare, automatizare
  • Construiești portofoliu — contribuțiile tale sunt publice pe GitHub
  • Ești parte din comunitate — alături de alți voluntari care cred în accesul universal la informație

Cum funcționează

  1. Propunere — un voluntar propune un scraper printr-un pull request
  2. Review — echipa verifică structura, selectoarele și corectitudinea datelor
  3. Testare — scraperul rulează în mediu de test, se validează job-urile extrase
  4. Activare — scraperul este aprobat și programat în GitHub Actions
  5. Rulare periodică — extrage automat job-uri noi și le trimite la API
  6. Monitorizare — se urmărește sănătatea scraperului și se alertează la eșec
  7. Întreținere — oricine poate actualiza un scraper când sursa se modifică

Acces universal la baza de date

peviitor.ro este o platformă deschisă. Baza de date poate fi accesată de oricine:

  • Prin API public — orice aplicație terță se poate conecta și citi datele (vezi EPIC-007)
  • Prin interfața web — utilizatorii caută și filtrează direct pe site
  • Prin exporturi — datele pot fi exportate în formate deschise (JSON, CSV)

Astfel, informația nu rămâne blocată într-un singur loc — ea circulă liber, îndeplinind misiunea de acces universal la oportunități de carieră.

Ciclul virtuos

Comunitatea scrie scrapere → Baza de date crește → Utilizatorii găsesc locuri de muncă → Tot mai mulți utilizatori → Tot mai mulți voluntari → Tot mai multe scrapere.

Fiecare contribuție, indiferent cât de mică, face parte din acest ciclu. Un scraper pentru o singură companie poate însemna zeci de job-uri pe lună pentru oameni care altfel nu le-ar fi găsit.

Feature-uri derivate

  • Scraper Template — șablon standard pentru orice limbaj
  • API Client Libraries — biblioteci pentru Python, PHP, Node.js
  • Scraper Validator — unealtă care verifică structura și datele
  • GitHub Actions Workflow — pipeline standard de scraping și submit
  • Scraper Health Dashboard — panel cu statusul tuturor scraper-elor
  • Alerts & Notifications — notificări pe Discord pentru eșecuri
  • Contributor Hall of Fame — recunoaștere publică a voluntarilor
